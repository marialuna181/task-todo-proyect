# task-todo-proyect
Proyecto curso React avanzado, utilizando redux, jsx

# clonar
git clone https://github.com/marialuna181/task-todo-proyect.git

# instalar
npm install

# iniciar
npm start

# build produccion
npm run build

# build development
npm run dev
